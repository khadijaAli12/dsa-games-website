## What is DSA Games?

DSA Games is not your average algorithm learning platform. It's a **gamified**, **interactive**, and **visual** playground where you can:

- **Master Sorting Algorithms**  
  Watch Bubble, Merge, Quick, Heap, and other sorts come to life with real-time visualizations!

- **Navigate Complex Mazes**  
  Explore graph algorithms using pathfinding and backtracking techniques.

- **Explore DSA Concepts**  
  Experience algorithm execution through hands-on learning ‚Äî no more dry theory!

- **Practice with Real Challenges**  
  Solve coding challenges and implement algorithms in our interactive playground.

##  Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/khadijaAli12/dsa-games-website.git
cd dsa-games-website
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5175` (or the port shown in your terminal)

The project implements a comprehensive design system using CSS variables:
